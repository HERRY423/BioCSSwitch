# BioCSSwitch bio-v0.4.2 — 蓝白科研工作台

BioCSSwitch 0.4.2 聚焦桌面研究工作台的可读性与操作效率，在不改变研究流程和模型连接行为的前提下，完成蓝白科研视觉升级与设置页重排。

## 界面与布局

- 工作台、研究轨道卡片、设置表单和任务编译对话框统一为蓝、白、浅蓝灰配色。
- 连接配置放在左侧，运行控制与高级设置放在右侧，减少常用操作之间的往返滚动。
- 科研工具包、隐私合规、MCP/Skill 验证和任务路由采用双栏模块布局。
- 小窗口自动回落为单列，保留完整的键盘焦点和响应式行为。

## 兼容性与质量

- 保留所有已有控件 ID、事件绑定、模型配置、工具包装配和研究任务启动逻辑。
- 应用与 Python 包版本统一为 `0.4.2`。
- macOS Release 工作流构建 Apple Silicon DMG；当前构建未经过 Apple 公证，首次启动可能需要右键选择“打开”。

## 研究使用声明

BioCSSwitch 是研究辅助工具，不提供医疗建议。AI 生成的证据综合、靶点排序和实验建议必须由合格研究者审阅和核验。
